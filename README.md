# Spring Boot Stripe Payment Integration

A simple and robust Spring Boot application integrating Stripe for seamless payment processing.

## Features
- Stripe Checkout Session integration
- Success and Cancel landing pages
- Product metadata handling
- REST API for checkout initiation

## Setup
1. Clone the repository.
2. Add your Stripe Secret Key to `src/main/resources/application.properties`.
3. Run with `./mvnw spring-boot:run`.

## Technologies
- Java 21
- Spring Boot 3.x
- Thymeleaf
- Stripe Java SDK
